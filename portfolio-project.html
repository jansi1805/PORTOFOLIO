About This Project

This is a personal portfolio website built with HTML, CSS, and JavaScript. It showcases my skills, projects, and contact information, and features modern web libraries like Animate.css, Typed.js, and Particles.js for enhanced interactivity and design.

Features

  • Responsive design for all devices
  • Animated hero section with typing effect
  • Interactive background using Particles.js
  • Skills bar chart and project showcase
  • Contact form with Formspree integration

Technologies Used

  • HTML5, CSS3, JavaScript
  • Animate.css
  • Typed.js
  • Particles.js

Preview

How It Works

The homepage introduces the developer, lists skills and projects, and provides a contact form. Each project can be viewed in detail by clicking its 'View Project' link, which leads to a dedicated project page like this one.
